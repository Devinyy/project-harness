# project-harness

This repository contains a reusable project harness template for frontend engineering work with AI agents.

## Contents

- `harness-templete-v1.0/project-root/agent.md`: agent operating instructions for a project.
- `harness-templete-v1.0/project-root/docs/`: project knowledge base templates, including project overview, architecture, domain model, coding standards, API contracts, UI patterns, task playbooks, troubleshooting, and risky areas.
- `harness-templete-v1.0/project-root/skills/`: task-oriented skill prompts for technical solution design, code generation, and frontend testing.

## Usage

Copy `harness-templete-v1.0/project-root` into a target project and replace placeholder values in the documents with real project information.

Start with:

1. `docs/00_INDEX.md`
2. `docs/01_PROJECT_OVERVIEW.md`
3. `agent.md`

Then fill the remaining documents according to the project context.
